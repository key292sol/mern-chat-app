# MERN Chat App

A Chat App created using the MERN stack.
You can create account and chat with others.

## Steps to run:

### For running server:

1. Go to `server` folder in terminal
1. Install dependencies using `npm install`
1. Create a file named `.env`
1. Add your server `PORT` and `MONGO_URL` in the `.env` file
1. Run the server using `npm start`

### For running client side:

1. Go to `frontend` folder in terminal
1. Install dependencies using `npm install`
1. Run the client using `npm start`
